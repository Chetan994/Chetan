# Iris Classifier Project

## Description
This is a simple machine learning project using the Iris dataset to classify flower species based on sepal and petal measurements.

## Features
- Data visualization with confusion matrix
- Logistic Regression model
- Live prediction example

## Requirements
Install required packages:
```
pip install numpy pandas matplotlib scikit-learn seaborn
```

## Running the Project
To execute the project:
```
python iris_classifier.py
```

## Output
- Model accuracy
- Classification report
- Confusion matrix plot
- Prediction of iris species from sample input
